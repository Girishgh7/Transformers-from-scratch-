# 🚀 Transformers from Scratch

This repository contains my **from-scratch implementation of the Transformer architecture**, inspired by the seminal paper *Attention Is All You Need* (Vaswani et al., 2017).  
The goal is to demonstrate a deep understanding of how Transformers work under the hood, without relying on pre-built libraries like Hugging Face or PyTorch’s `nn.Transformer`.

---

## 📌 Objectives
- Implement the **core building blocks** of the Transformer:
  - Scaled Dot-Product Attention
  - Multi-Head Attention
  - Positional Encoding
  - Feedforward Networks
  - Encoder & Decoder Modules
- Train and evaluate the model on toy datasets to validate correctness.
- Provide clean, modular, and well-documented code for learning and reproducibility.

---

## 🛠️ Features
- **Minimal dependencies**: Built using PyTorch and NumPy.
- **Educational focus**: Each component is implemented step by step, with clear explanations.
- **Colab-ready notebooks**: Easy to run and experiment with.
- **Interpretability tools**: Visualizations of attention weights for better understanding.
